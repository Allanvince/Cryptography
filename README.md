# Cryptography Assignment  

This repository contains a **Cryptography Assignment** related to **ICS 2411 - Computer Security and Cryptography**. The assignment explores various cryptographic concepts, including encryption, decryption, hashing, and security protocols. It serves as a valuable resource for students and researchers interested in understanding the fundamentals of cryptography.  

## Contents  
- **Cryptography Assignment PDF**: A detailed document covering key cryptographic principles and techniques.  
- **PowerPoint Presentation**: A summarized version of the assignment in slide format (limited view).
